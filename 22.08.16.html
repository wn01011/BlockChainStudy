style : 속성, 그 태그의 크기, 외곽선, 위치, 배경 등을 설정한다. 기초적인 방법 웹페이지 처음은 html 파일로 주고 받았어. 그런데 html 파일이 너무 커져서 보기가 어렵다. CSS(Cascading Style Sheets) 파일로 꾸미는 기능이 분화가 되어 나오게 되었다.

너비나 높이나 숫자를 이용한 수치들에 대한 단위 : px, %, em, rem, vw, vh;

px : 픽셀 한칸.

% : 부모를 기준으로한 비율, 100% = 부모의 너비랑 같음.

em : 글씨 크기를 기준으로 한다. 부모의 글씨 크기

rem : 글씨 크기를 기준으로 한다.html 의 글씨 크기. 많이 사용한다.

ex) 글씨 크기가 16px, 그럼 1em, 1rem은 16px 글씨 크기가 32px, 1rem = 1em 32px, 0.5rem = 16px

vw, vh : viewport, 브라우저의 크기나 스마트폰의 크기를 기주으로 %, 100vw = 화면의 너비, 100vh 화면의 높이

width : 영역의 너비 max-width : 영역의 최대 너비 min-width : 영역의 최소 너비 %, vw, vh은 기준이 있고 그 기준의 비율로 너비를 설정함.

어떤 내용을 넣었는데, 최소 100px은 있어야 제대로 정보 전달이 될 때, min-width를 쓴다.
어떤 내용을 넣었는데, 너무 크면 안된다면 max-width를 쓴다.

height : 높이
max-heigth : 영역의 최대 높이
min-height : 영역의 최소 높이
